SOFTWARE LIBRE

Escritorio

7-Zip

7- zip es un compresor/descompresor de archivos que
maneja los formatos ZIP, RAR, CAB, GZIP, BZIP2 y TAR
asi como un formato propio

Audacity

Audacity es un editor de audio. Permite reproducir,
grabar, editar, mazclar, aplicar efectos, etc

GIMP

GIMP (GNU image Manipulation Program) es un
programa de retoque fotografico y composicion y edicion
de imágenes

Inkscape

Inkscape es un editor de gráficos vectoriales que utiliza
el formato SVG y que tambien importa formatos como
Postscript, EPS, JPEG, PNG y TIFF y exporta un PNG y
varios formatos vectoriales

LibreOffice

LibreOffice es un suite ofimática completa compuesta
de procesador de textos, hoja de calculo,
prosentaciones, programa de dibujo y base de datos y
editor de escuaciones

Internet

FileZilla

FireZilla es un cliente de FTP

Firefox

Firefox es un navegador web altamente personalizable y
ampliable mediante los cientos de extensiones y temas
disponibles

HTTrack

HTTrack permite descargar sitios webs completos para
su consulta online

Pidgin

Pidgin es un cliente de mensajeria instantanea
compatible con las redes AIM (Ptrococolos Oscar y TOC)
ICQ, MSN Messenger, Yahoo, IRC, Jabber, Gadu-Gadu, Zephyr y otras

Thunderbird

Thunderbird es un cliente de correo de los creadores de
Firefox. Incorpora multitud de caracteristicas avanzadas y, como Firefox, puede personalizarse, mediante temas y
ampliar sus posibilidades mediante extensiones
